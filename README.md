# Virgin Money - Android technical assignment
## By Jason Calvert 
Virgin Money is a directory app which allows the staff to view their collegues contact details and also see which
rooms are currently occupied.</br>The Virgin Money App is a kotlin based Android Application which </br>was built using the best practices and the latest
technologies.
</br>

This feature displays a list of employees in a recyclerview which makes it easy to display large sets of data. The recyclerview does what it says on the tin,</br>it recycles the views as they scroll onto the screen, which means that the views are not destroyed but recycled for new items.
